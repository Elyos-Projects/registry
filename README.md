# Elyos registry

The public index of approved Elyos good-deed projects.

This is the canonical source that `elyos browse` reads to discover projects. Each entry points
at a project repo under the [`Elyos-Projects`](https://github.com/Elyos-Projects) org, where the
project's own tasks live under `tasks/`.

- **File:** [`registry.json`](./registry.json)
- **Raw URL:** `https://raw.githubusercontent.com/Elyos-Projects/registry/main/registry.json`

To get listed, open a PR adding your project entry (see the project schema in the platform repo).

## What Elyos is

Elyos turns spare AI capacity into open-source good deeds. A contributor pulls a
community-approved task, has their own coding agent do the work, and opens a PR.
